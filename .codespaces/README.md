# 🚀 GitHub Codespaces에서 실행하기

이 프로젝트는 GitHub Codespaces에서 바로 실행할 수 있습니다!

## 사용 방법

1. **Codespaces 시작**
   - 저장소 페이지에서 "Code" 버튼 클릭
   - "Codespaces" 탭 선택
   - "Create codespace on main" 클릭

2. **자동 설정**
   - Codespace가 생성되면 자동으로 패키지가 설치됩니다
   - 터미널에서 `python app.py` 실행

3. **포트 포워딩**
   - Codespace가 자동으로 포트 5000을 포워딩합니다
   - "Ports" 탭에서 공개 URL 확인
   - 또는 알림에서 "Open in Browser" 클릭

4. **접속**
   - 생성된 URL로 접속하면 바로 사용할 수 있습니다!

## 팁

- Codespace는 무료로 사용할 수 있습니다 (월 60시간)
- 여러 명이 동시에 사용할 수 있습니다
- 코드 수정이 즉시 반영됩니다
